Server:
loop:
Verifica se há espaço para novos envios na janela: windowFull
Lê buffer do arquivo
Calcula hash de detecção de erro.
Cria cabeçalho UDP de 64 bits: hash (32) + número de sequência (32)
Envia cabeçalho + buffer
Bufferiza tudo para funcionamento da janela: windowInsert
Inicializa temporizador de 1s.
Incrementa o número de sequência.
Incrementa a contagem de bytes.
Recebe possíveis acks (64 bits).
Aplica acks: acknowledge
Desliza a janela, se preciso: removeAckds
fim do loop.

Client:
Inicializa a janela, com números de sequência de 0 a tam_janela-1 e todos os Rcvd = 0.
loop:
Recebe pacote do server
Faz detecção de erro.
Lê número de sequência.
Buferiza pacotes recebidos corretamente.
Atualiza janela e lastRcvd (último pacote recebido na ordem correta).
Desliza janela, se possível.
Envia Ack com número de lastRcvd.
fim do loop
